## About

This organization hosts the hands-on companion materials for **_The Essential IoT Integration
Handbook for Smart Construction_**, a practitioner- and student-focused guide to designing,
building, deploying, and operating Internet-of-Things systems on construction sites.

The handbook is built around a single idea: you learn an IoT system best by **building one**. It takes
the reader from a first temperature reading on a benchtop sensor all the way to a deployed, accepted,
and maintained monitoring system — the **Smart Jobsite Cloud Gateway**.

## The book

*The Essential IoT Integration Handbook for Smart Construction*
**Authors:** Aaron Costin · Kyle Morman · Janise McNair
**Publisher:** Elsevier · *(edition / ISBN / year — forthcoming)*

A ten-chapter, practitioner-oriented handbook that pairs the concepts of construction IoT —
devices and sensors, wireless networking, data management, deployment planning, real-time
monitoring, and system operations — with a continuous, hands-on project carried from the first
chapter to the last.

## A three-tier, hands-on approach

The handbook teaches through one connected project, layered across three tiers so readers can engage
at the depth that suits them:

- **Chapter Activities** — a worksheet-style design exercise in every chapter, written to be
  vendor-neutral and durable.
- **Appendix Labs** — concrete, step-by-step builds using commodity hardware and open tools, each one
  picking up where the last left off.
- **Companion code** — a living repository of the sketches and host scripts that bring each lab to
  life, released alongside the book.

The project grows one dependable step at a time:

| Stage | What the reader builds |
|---|---|
| First reading | A sensor node reporting temperature and humidity over serial |
| Structured data | A checked, structured payload the host can trust |
| To the cloud | A node → gateway → cloud path to a live dashboard |
| Install & accept | A commissioning checklist and a measured acceptance test |
| Alerts & control | Threshold alerts, plus local safety that survives a network loss |
| Operations | Health monitoring that flags a silent node |
| **Capstone** | The complete **Smart Jobsite Cloud Gateway**, assembled from every part |

## The stack

Commodity, widely available, and approachable for newcomers:

- **Hardware:** Arduino / ESP32-class boards with a low-cost temperature–humidity sensor
- **Host:** small, dependency-light Python scripts
- **Cloud:** a free-tier cloud dashboard, with an open-source alternative noted throughout

## Who it's for

Construction professionals exploring IoT for the first time, students in construction, civil, and
related engineering programs, and educators looking for a project-based path into connected
construction systems.
